# ❤️ LDR Memory Booth

### Two places. One memory.

**LDR Memory Booth** is a virtual photobooth experience designed for long-distance couples. It allows two people separated by distance to create, capture, and preserve memories together through a digital photobooth.

The goal is to recreate the feeling of a real mall photobooth while allowing couples to create shared moments from different locations.

---

# 📸 Project Preview

> A digital photobooth where couples can create a room, invite their partner, take photos together, and generate a memory strip.

---

# ✨ Features

## 🏠 Couple Room System

* Create a private couple room
* Generate a unique invitation code
* Join a room using an invitation code
* Couple lobby before starting the session

## 📷 Virtual Photobooth

* Webcam integration using browser camera API
* Countdown before capturing
* 6-photo memory session
* Photo preview generation
* Photobooth-style experience

## ❤️ Memory Strip Generator

* Combine captured photos into a vertical photo strip
* Add LDR branding
* Download final memory image
* Ready for printing and sharing

## 🎨 Themes

Available themes:

* ❤️ Love
* 🌸 Sakura
* 🌙 Midnight
* 🎞 Retro

More themes will be added in future updates.
